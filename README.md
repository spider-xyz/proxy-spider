* 测试地址: (单机勿压)
```shell
http://139.198.191.82:5000/count

```

### 下载安装

* 下载源码:

```shell
git clone https://github.com/spider-xyz/proxy-spider.git
```

* 安装依赖:

```shell
pip3 install -r requirements.txt
/usr/bin/nohup python3 run.py >> /tmp/proxy.log 2>&1 &
```
